    • First gets input from the terminal

    • Then runs the prime finder method

    • Then runs the method that builds unique combos

    • Finally tests combos against secret() for additive

    check_if_additive(x, y) click to toggle source

    Compares if additive passing two int (from #test_prime_combos_for_additive) As long as each check remains true, program continues onto next check with next combo Upon first false - program informs user which prime combo failed and terminates

    # File additive_prime.rb, line 53
def check_if_additive(x, y)
        unless secret(x+y) == secret(x) + secret(y)
                puts "Uh oh"
                puts "For Prime combo #{@iteration.join(" and ")}"
                abort("The secret method has been proven NOT additive.")
        end
end

    test_prime_combos_for_additive() click to toggle source

    For each prime combo - runs #check_if_additive Passes in both values in the combo Saves iteration for error message in #check_if_additive If all checks pass - success message displays and program terminates

    # File additive_prime.rb, line 65
def test_prime_combos_for_additive
        puts "Now checking whether the secret function is additive."
        @prime_combos.each do |a|
                @iteration = a
                check_if_additive(a[0], a[1])
        end
        puts "Hurray! We made it through the checker."
        puts "That means for all primes less than #{@n}: the secret function has been proven additive!"
end
